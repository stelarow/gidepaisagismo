# 🌿 GIDI Paisagismo - Resumo Executivo do Projeto

## 📊 Visão Geral

Site profissional e sofisticado desenvolvido para **GIDI Paisagismo**, combinando design moderno com funcionalidades avançadas para atrair e converter clientes.

---

## ✨ Destaques do Projeto

### 🎨 Design Premium
- **Inspirações**: Mayara Schaden (minimalismo) + Telhado Vivo (sustentabilidade)
- **Paleta**: Verde floresta (#2d5f3f), Verde vivo (#78b719), Terra (#a67c52)
- **Tipografia**: Playfair Display (elegante) + Inter (moderna) + Montserrat (impacto)
- **Estilo**: Sofisticado, clean, com elementos naturais

### 🚀 Funcionalidades Principais

1. **Hero Section Full-Width**
   - Banner impactante 100vh
   - Overlay gradient elegante
   - Call-to-action destacado
   - Animação cross-fade

2. **Portfólio Dinâmico**
   - Galeria com filtros por categoria
   - Lightbox para visualização ampliada
   - Grid responsivo masonry
   - Lazy loading otimizado

3. **Calculadora de Orçamento** ⭐
   - Estimativa em tempo real
   - Slider de complexidade
   - Campos dinâmicos
   - Faixa de preço automática

4. **Galeria Instagram**
   - Grid 4 colunas responsivo
   - Hover effects sofisticados
   - Integração API/Widget/Estático
   - Link direto para perfil

5. **Formulário Inteligente**
   - Validação em tempo real
   - Formatação automática de telefone
   - Anti-spam (honeypot)
   - Integração FormSpree pronta

6. **Depoimentos em Carrossel**
   - Swiper.js com autoplay
   - Navegação touch-friendly
   - Sistema de avaliação (5 estrelas)
   - Depoimentos rotativos
