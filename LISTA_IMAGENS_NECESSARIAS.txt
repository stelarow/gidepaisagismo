✅ DICAS IMPORTANTES
═══════════════════════════════════════════════════════════════════

1. QUALIDADE DAS IMAGENS:
   - Use imagens de alta resolução
   - Boa iluminação
   - Cores vibrantes
   - Evite imagens borradas ou pixeladas

2. OTIMIZAÇÃO:
   - Comprima as imagens antes de usar
   - Use https://tinypng.com (gratuito)
   - Mantenha qualidade, mas reduza tamanho do arquivo

3. FORMATO DOS ARQUIVOS:
   - JPG para fotos (menor tamanho)
   - PNG para logo (mantém transparência)
   - WebP se possível (melhor compressão)

4. NOMES DOS ARQUIVOS:
   - Use exatamente os nomes sugeridos
   - Sem espaços ou caracteres especiais
   - Tudo em minúsculas
   - Use hífen (-) ao invés de underscore (_)

5. DIREITOS DE IMAGEM:
   - Certifique-se de ter permissão para usar
   - Fotos de clientes: obter autorização
   - Fotos de projetos: documentar permissões
